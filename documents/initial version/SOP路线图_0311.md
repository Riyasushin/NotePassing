git push origin main# 塞纸条 MVP 50h 开发路线图
Android BLE Nearby Social App

目标：  
在 **50小时内完成一个可以真实运行的 MVP**，具备：

- 用户登录
技术路线：

  - BLE：附近发现
  - 网络：聊天和用户信息

核心原则：

> **让 AI 每次只完成一个小模块，不要让 AI 一次做完整 App**

---
工作模式：
用 Android Studio 新建并运行项目

用 Cursor 打开同一目录

让 Cursor 分模块生成代码

复制/接受修改后回 Android Studio 编译

报错直接把 Logcat + 报错栈 + 相关文件 丢给 Cursor 修

BLE 一律真机调试，不折腾远程安卓开发
