# 塞纸条 (NotePassing)

如果我们把社交媒体空间与真实的物理空间同步，会发生什么？

**塞纸条** 是一个 Android App，用于实现 **近距离现实社交网络**。

用户在同一物理空间中，可以通过手机蓝牙发现彼此并即时互动。

适用场景：

- 校园
- 会议
- 漫展
- 聚会
- 搭讪
- 社交活动

核心理念：

> 将现实空间同步到社交网络层，让同一物理空间中的人可以即时互动。

---

# MVP 功能

## 1. 附近用户发现

客户端通过 **BLE 广播临时加密 ID**。

用户打开 App 后可以看到：

- 附近用户
- 昵称
- 标签
- 简短简介
- 估计距离

名片信息包含：

- nickname
- avatar
- tags
- profile

---

## 2. 短距聊天

用户可以向蓝牙范围内的用户发送 **短距即时聊天**。

规则：

- 对方未回复前最多发送 **2 条消息**
- 若双方超出蓝牙范围 **1 分钟**，聊天窗口失效

---

## 3. 成为好友

在短距聊天中可以发送 **好友申请**。

双方同意后建立 **好友关系**。

好友之间可以：

- 进行长期聊天
- 不再受蓝牙距离限制

---

## 4. 熟人再次接近

当好友重新进入蓝牙范围时：

- 触发 **Boost 提示**

例如：

- 震动
- UI 高亮

用于增强线下社交体验。

# 技术原理

蓝牙仅用于 **发现附近用户**。

所有用户信息与聊天数据均通过服务器传输。

工作流程：

1. 客户端 BLE 广播 **临时加密 ID**
2. 其他设备扫描到该 ID
3. 客户端将 ID 上传服务器
4. 服务器返回对应用户名片
5. 用户可以发起聊天或好友申请
